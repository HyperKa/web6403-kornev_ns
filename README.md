Для лабораторной работы 4 реализовано:
Модернизация веб-сайта с использованием современных подходов к верстке (Flexbox, Grid), стилизации (CSS Variables, анимации) и объектно-ориентированного программирования на JavaScript.
*   **Верстка с помощью Flexbox**: Главная страница (`index.html`) сверстана с использованием Flexbox для позиционирования элементов в навигации, в блоке-герое и в секции с преимуществами.
*   **Верстка с помощью Grid**: Страницы "Врачи" (`doctors.html`) и "Контакты" (`contacts.html`) используют CSS Grid для создания адаптивной двумерной сетки карточек и основной разметки страницы.
*   **CSS Variables**: Все цвета на сайте заданы через CSS-переменные в блоке `:root`, что обеспечивает централизованное управление темой и легкую смену палитры.
*   **CSS Анимация**: На странице контактов реализована комплексная CSS-анимация "пульсирующее сердце", состоящая из двух `@keyframes` правил (`pulse` и `wave`) и использующая `animation-delay` для создания эффекта расходящихся волн.
